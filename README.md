# Gallery Images

This folder contains images for the furniture gallery.

## Image Naming Convention

Please add your furniture images using the following naming convention:
- sofa1.jpg, sofa2.jpg, sofa3.jpg (for sofas)
- bed1.jpg, bed2.jpg, bed3.jpg (for beds)
- chair1.jpg, chair2.jpg, chair3.jpg (for chairs)
- table1.jpg, table2.jpg, table3.jpg (for tables)
- cabinet1.jpg, cabinet2.jpg (for cabinets)
- wardrobe1.jpg, wardrobe2.jpg (for wardrobes)
- desk1.jpg, desk2.jpg (for desks)

## Recommended Image Specifications

- Format: JPG or PNG
- Dimensions: 800x800px minimum (square format recommended)
- File Size: Keep under 500KB per image for optimal loading

## Current Gallery Structure

The Gallery component will automatically load images from this directory.
If an image is not found, it will display a placeholder with the category name.
